# Casper Agentic Buildathon Submission Draft

## Project Name

CasperSect AI

## Tagline

Agentic wallet-risk intelligence with verifiable scan records on Casper Testnet.

## Short Description

CasperSect AI is an agentic Web3 security platform that analyzes wallet behavior, generates AI-powered risk scores, recommends actions, and records scan evidence on Casper Testnet through an Odra smart contract.

## Problem

Wallet users, DeFi protocols, and RWA applications often lack simple, explainable, and verifiable wallet-risk signals before interacting with an address.

## Solution

CasperSect AI turns wallet analysis into an autonomous agent workflow:

1. Analyze wallet behavior.
2. Score risk.
3. Explain suspicious patterns.
4. Recommend next actions.
5. Record scan evidence on Casper Testnet.

## Buildathon Alignment

- Agentic AI risk recommendation system
- Casper Wallet integration
- Odra smart-contract registry
- Casper Testnet transaction flow
- DeFi/RWA applicability for compliance, monitoring, and counterparty risk

## Tech Stack

- Frontend: Next.js 15, TypeScript, Tailwind CSS
- Backend: Node.js, Express, TypeScript, OpenAI optional
- Blockchain: Casper Testnet, Casper Wallet, Odra smart contracts
- AI: deterministic risk engine with optional LLM-based recommendation layer

## Demo Flow

1. Connect Casper Wallet.
2. Run wallet risk scan.
3. Review AI explanation and recommendation.
4. Generate evidence hash.
5. Record scan on Casper Testnet.
6. Show deploy hash and scan result.
